### Сайт для спросмотра прогноза погоды
Над проектом работал Епифанов Кирилл Антонович, ИСУ 395713, группа К3343, поток WEB 3.2.

Сайт представляет из себя страницу, где пользователь может выбрать несколько городов или текущее местоположение и получить по ним погодные данные

###### Доступен следующий функционал:
- Добавление текущего города по геолокации браузера
- Добавление произвольного города
