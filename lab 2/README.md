### Сайт для создания заметок
Над проектом работал Епифанов Кирилл Антонович, ИСУ 395713, группа К3343, поток WEB 3.2.

Сайт представляет из себя страницу, где пользователь может создать список задач.
Тело сайта полностью создается через JS и DOM. 
Адаптивность реализована через CSS. 

###### Доступен следующий функционал:
- Создание задачи с пользовательским названием и датой
- Редактирование названия, статуса и даты задачи
- Удаление задачи
- Сортировка задач по дате
- Изменение порядка задач перетаскиванием
- Фильтрация по названию и статусу задачи
