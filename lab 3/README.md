### Сайт для создания заметок
Над проектом работал Епифанов Кирилл Антонович, ИСУ 395713, группа К3343, поток WEB 3.2.

Сайт представляет из себя страницу, где пользователь может играть в 2048

###### Доступен следующий функционал:
- Сохранение результата с кастомным именем пользователя
- Функционал игры 2048
- Брать ход назад
- Начинать новую игру
